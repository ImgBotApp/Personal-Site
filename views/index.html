And I am a math geek.

I find math to be spectactular. Everything in this world, literally everything, can be translated into a mathematical statement. "Mathematics is the language with which God wrote the Universe." as Galileo Galilei once said. I feel that the barrier that most people run into is believing that math is a big collection of numbers and just doing tons of number crunching, but that couldn't be farther from the truth. Math is much more than algebra, geometry, and calculus. Mathematics is how Google Maps can find the directions from point A, from anywhere in the world, to point B using a branch of mathematics called Graph Theory. Cryptography is the mathematical study of encryptions and secret communication. Interactions between players in a board game or video game is studied through a mathematical branch called Game Theory. Without math, we wouldn't have any of the phenomenal advances in technology and living that we currently enjoy. We wouldn't have the physics necessary to create planes and cars or fly to the moon. We wouldn't have the computers that do so much for us. Even in "natural" non-technological items we can find math. The harmonics on stringed instruments is simply a series of fractal lengths on the string that produce natural frequencies. A fibonacci spiral can be observed in many common flowers. Math is beautiful, complex, and essential.

Jeopardy

This project, Jeopardy, was my Final Project from my Independent Study Computer Science course in high school. It is to this date the largest project I have coded strictly by myself under the supervision of my teacher Joel Mellen. Totalling 1500+ lines, this application is a fully functioning and robust version of Jeopardy, allowing you to make your own questions and rounds, which are saved into text files. You can then choose which rounds to play, select teams or individual, and how long your question timer is (15, 30, or 45 seconds) Listed below are all the features the Jeopardy project entailed:

  • Select a background color for the general application
  • Select the length of the question timer (15, 30, or 45 seconds)
  • Multithreading to run the question listener and timer concurrently
  • File saving/outstream to save the questions and rounds into text files
  • File reading/instream to read the files and pull the questions and answers to be played in the game
  • Fully Graphical User Interface
  • Background music of the Jeopardy theme (also running concurrently)

Projects & Ideas



I love personal projects because they allow me to expand my knowledge of computer science and software engineering in areas that I am particularly interest in, or take a break for CS courses at school by taking part in relaxing side-projects such as web-design and game development

Project Euler is probably (in my opinion) the best tool to create logical, witty, and efficient programmers. It is a large collection of math-based questions and puzzles for which you must write a short program to solve. I love these kind of problems because it doesn't test your knowledge of a certain technology, but your problem-solving skills which I believe to be far more valuable. If you can solve these problems in a programming language, chances are you will have no problem picking up other languages as needed. My high school computer science teacher told me that once you know one language, you know them all. And he's correct, but he's not referring to syntax. He is talking about the logic and programming concepts such as boolean operations, loops, if-statements, etc. Knowing how to implement these basic tools to create complex logic is a powerful tool. Hopefully after this semester ends I'll have much more time to get back on this.
